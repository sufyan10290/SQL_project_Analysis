###   1. Top Paying Data Analyst Jobs 
To identify the highest paying roles, I filtered the data Analyst positions by average yearly salary and location, focusing on remote jobs. The Query below highlights the high paying Data Analyst rolesfor the Year 2023 

```sql
SELECT job_id,
    name AS company_name,
    job_title_short,
    job_location,
    salary_year_avg
FROM job_postings_fact
JOIN company_dim USING(company_id)
WHERE 
    job_title_short = 'Data Analyst'
    AND salary_year_avg IS NOT NULL 
    AND job_work_from_home = TRUE
ORDER BY salary_year_avg DESC
LIMIT 5;
```
Here is the breakdown of the top Data Analyst jobs in 2023:
- **Wide Salary Range :** Top 10 paying Data Analyst roles span from $184,000 to $650,000 indicating significant salary potential in the field.
- **diverse Employers :** Companies like SmartAssert, Meta and AT&T are among those offering high salries, showing a broad interest across different industries.
- **Job Title Variety :** there is a high diversity in job titles from Data Analyst to director of Analytics, reflecting Varied roles and specialization within Data Analytics.

### 2. Skills for top paying jobs
To understand what skills are required for the top paying jobs, I created a CTE of the top 10 paying Data Analyst jobs with the company names provided, then I joined the `skills_job_dim` and the `skill_dim` table which had the skills data.   
```sql
WITH top_paying_roles AS(
        SELECT *
        FROM job_postings_fact jpf
        JOIN company_dim USING(company_id)
        WHERE 
            job_title_short = 'Data Analyst'
            AND salary_year_avg IS NOT NULL 
            AND job_work_from_home = TRUE
        ORDER BY salary_year_avg DESC
        LIMIT 10)
SELECT 
    name AS company_name,
    job_title_short,
    job_location,
    salary_year_avg,
    skills
FROM top_paying_roles tpr
JOIN skills_job_dim sjd
    USING(job_id)
JOIN skills_dim sd 
    USING(skill_id)
ORDER BY salary_year_avg DESC;
```
here is the breakdown of the most in demand skills of the top 10 highest paying Data Analyst Roles in 2023.
- Sql leads with a bold count of 8
- Python came closely with a count of 7
- Tableau is the next highly sought after with count of 6
- Other skills such as R, snowflakes, pandas and Excel show varying degree of demand.

### 3 Top Demanded Skills  
This query focused on the top skills that are most likely to appear on a Data Analyst job posting thus high demand and most requested skills.  
```sql
SELECT 
    sd.skill_id,
    sd.skills,
    COUNT(*) AS skill_count
FROM job_postings_fact jpf
JOIN skills_job_dim sjd
    USING(job_id)
JOIN skills_dim sd 
    USING(skill_id)
WHERE 
    job_title_short = 'Data Analyst'
    --AND salary_year_avg IS NOT NULL 
    AND job_work_from_home = TRUE
GROUP BY sd.skill_id,sd.skills
ORDER BY skill_count DESC
LIMIT 10;
```
Here is the breakdown of the most demanded skills for data analyst roles in 2023:
- **SQL and Excel** remain fundamental, emphasizing on the need for strong foundational skills in data processing and spreadsheet manipulation.  
- **Programing and Visualization Tools** like **Python**, **Tableau** and **Power BI** are essential, pointing towards the increasing importance of technical skills in data story telling and decision support.

### 4. Skills based on Salary
We also explored the average salaries associated with different skills revealing which skills are the highest paying. this was achieved simply by replacing the `COUNT` aggregation function with the `Average` aggregation funtion performed on the salary_year_avg in the prevoius query that did the count.  
```sql
SELECT 
    sd.skill_id,
    sd.skills,
    ROUND(AVG(salary_year_avg),0) AS avg_skill_salary
FROM job_postings_fact jpf
JOIN skills_job_dim sjd
    USING(job_id)
JOIN skills_dim sd 
    USING(skill_id)
WHERE 
    job_title_short = 'Data Analyst'
    AND salary_year_avg IS NOT NULL 
    AND job_work_from_home = TRUE
GROUP BY sd.skill_id,sd.skills
ORDER BY avg_skill_salary DESC
LIMIT 25;
```  
Here is a breakdown of the results from the top 25 skills of Data Analyst based on salary.  
- **High Demand for big Data & ML Skills** Top salaries are commanded by analyst skilled in big data technologies(pystack, Couchbase), machine learning tools(DataRobot,Jupyter), python libraries (Pandas, Numpy) reflectingthe industry's high valuation of data processing and predictive modeling capabilities.
- **Sofware Development & Deployment Profeciency :** Knowledge in development and deployment tools(Gitlab, Kubanetes, Airflow) indicates a lucrative cross over between data analysis and Engineering, with a premium on skills that facilitate automation and effecient data pipeline management.  
- **Cloud Computing Expectise :** Familiarity with cloud and Data Engineering tools (Elasticsearch, Databricks,GCP) underscores the growing importance of cloud-base analytics environment, suggesting that cloud proficiency significantly boost earning potential in data analytics.

### 5 Most Optimal Skills to Learn  
By combinig both the `COUNT` and `AVERAGE` aggregation into the previous query, we are able to relate the count of each skill to its average salary. this was to pinpoit skills that are in high demand and have high salaries, offering a strategic focus for skill development.  
```sql
SELECT 
    sd.skill_id,
    sd.skills,
    COUNT(*) AS skill_count,
    ROUND(AVG(salary_year_avg),0) AS avg_skill_salary
FROM job_postings_fact jpf
JOIN skills_job_dim sjd
    USING(job_id)
JOIN skills_dim sd 
    USING(skill_id)
WHERE 
    job_title_short = 'Data Analyst'
    AND salary_year_avg IS NOT NULL 
    AND job_work_from_home = TRUE
GROUP BY sd.skill_id,sd.skills
HAVING COUNT(*) > 10 
    AND AVG(salary_year_avg) > 100000
ORDER BY avg_skill_salary DESC
LIMIT 10;
```
